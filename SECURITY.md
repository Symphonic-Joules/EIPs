# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Ethereum EIPs repository, please report it by emailing security@ethereum.org. Alternatively, you may use the GitHub Security Advisory feature for private disclosure.

We aim to acknowledge your report within 5 business days and provide a status update within 10 business days.

We support coordinated disclosure and will work with you to resolve the issue before any public disclosure. Please do not disclose vulnerabilities publicly until they have been addressed.

Relevant security issues for this repository include vulnerabilities in EIP specifications, issues affecting the EIPs website, or any other security concerns that could impact the integrity or availability of EIP content.
